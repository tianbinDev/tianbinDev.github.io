属性动画

发表于 2015-12-24   |   分类于 Android

与补间动画的区别

  • 补间动画只能为View实现动画效果，属性动画可以为任意对象添加动画效果
  • 补间动画通过ParentView不断改变ChildView的方式来实现动画效果，属性动画是通过改变对象属性的方式实现动画效果
  • 补间动画通过相关Api来实现对应的动画，属性动画中ObjectAnimator需要传入动画名称参数，进而调用动画名称对应的get，set方法实现动画效果

ValueAnimator

ValueAnimator的功能：

  • 计算动画开始值到借宿值的过渡
  • 设置播放次数，播放模式
  • 监听动画的执行

注：ValueAnimator的ofFloat方法允许传入多个参数

AnimatorSet，Animator

Android Api 提供了AnimatorSet来实现组合动画，AnimatorSet中的play方法会返回一个AnimatorSet.Builder实例，该实例提供了以下几个方法：

  • after(Animator anim) 插入动画到现有动画之后执行
  • after(long delay) 延迟指定毫秒执行现有动画
  • before(Animator anim) 插入现有动画到传入的动画之前执行
  • with(Animator anim) 同时执行现有动画和传入的动画

TypeEvaluator

Api中提供了TypeEvaluator接口来控制startValue到endValue过渡的快慢，可以通过重写evaluate方法的实现算法来改变动画的执行速度

  • fraction 表示当前动画的执行进度
  • startValue 表示初始值
  • endValue 表示结束值
