Android 开源代码干货

发表于 2016-11-07   |   分类于 Android ， OpenSource

Android 开源代码干货，包括动画／自定义View／开源项目等，持续更新中~

APP

  • 一个为控件添加深度效果的库
    https://github.com/danielzeller/Depth-LIB-Android-
  • Android 模版项目 使用到了 RxJava, Dagger 2, Espresso, Mockito 等库
    https://github.com/hitherejoe/Android-Boilerplate
  • Material Design 风格的登陆功能
    https://github.com/unipiazza/unipiazza-android-twostepslogin
  • 一款精致的Gank.io 客户端
    https://github.com/JohnnyShieh/Gank

Animation

  • Yalantis 出品：搜索过滤器交互效果
    https://github.com/Yalantis/SearchFilter
  • 让你的 App 不再那么无聊和单调，气泡风格的过渡动画效果，酷酷的！
    https://github.com/Cleveroad/BubbleAnimationLayout
  • 弹出分享控件动画，超赞
    https://github.com/JeasonWong/FlipShare
  • 仿支付宝芝麻信用动画效果
    https://github.com/HotBitmapGG/CreditSesameRingView
  • Shape Ripple is a library that emulates a ripple like animations with cool tweaks on the go. It runs on API level 11 and upwards
    https://github.com/poldz123/ShapeRipple
  • Material Design 效果的注册过渡动画
    https://github.com/JeasonWong/SignUpTransition
  • 发送信息时显示的过度动画（纸飞机）
    https://github.com/Jaouan/Sending-Animation-Example
  • https://github.com/lovejjfg/Circle

Button

  • 可以设置圆角的Button，同时实现了按钮点击时背景色渐变
    https://github.com/czy1121/roundbutton
  • 带有展开动画的 button , very cute
    https://github.com/daniel-martins-IR/MagicButton

CalendarView

  • 一个实现日历与列表交互的控件
    https://github.com/Kelin-Hong/CalendarListView
  • Material Design 风格的日历控件
    https://github.com/prolificinteractive/material-calendarview

Dialog

  • 一个简单，强大的广告活动弹窗控件
    https://github.com/yipianfengye/android-adDialog

FilePicker

  • 一款轻量的文件选择器，同样支持选择多种类型文件
    https://github.com/fishwjy/MultiType-FilePicker
  • 一款支持从页面底部拉起的图片选择器
    https://github.com/ParkSangGwon/TedBottomPicker
  • 自定义文件管理器
    https://github.com/Angads25/android-filepicker
  • 支持选择多种类型文件的选择器
    https://github.com/DroidNinja/Android-FilePicker

HighLightShowContent

  • 支持语法高亮的代码展示控件
    https://github.com/Softwee/codeview-android

ImageView

  • Android 高性能实时模糊（Blur）

    https://github.com/hugeterry/PloyFun
  • 仿即刻首页滚动效果
    https://github.com/JeasonWong/JikeGallery

  • TextView和ImageView等待数据返回时显示的加载动画
    https://github.com/elye/loaderviewlibrary
  • 仿支付宝 下拉加载动画
    https://github.com/andyxialm/SmileyLoadingView

  • Material Design风格的底部导航栏，效果类似Android Support Library 中的BottomNavigationView 控件
    https://github.com/roughike/BottomBar
  • UI优美的底部导航栏
    https://github.com/armcha/Space-Navigation-View
  • 一个以Material Design风格展示侧面导航栏的控件
    https://github.com/andremion/Floating-Navigation-View

Progress

  • 一个圆形进度条，控件中可以设置文字
    https://github.com/ldoublem/RingProgress
  • A DownloadProgressButton with Animation for Android
    https://github.com/cctanfujun/ProgressRoundButton
  • A ‘Google Fit’ like activity indicator for Android
    https://github.com/dlazaro66/WheelIndicatorView

RecyclerView

  • 支持多选的 RecyclerView
    https://github.com/dvdciri/MultiChoiceRecyclerView
  • RecyclerView 实现流式布局，可以用来做 Tag 标签效果
    https://github.com/BelooS/ChipsLayoutManager
  • 支持快速快动的RecyclerView
    https://github.com/L4Digital/FastScroll
  • RecyclerView snapping (实现列表第一个Holder总是完整显示，当滑动使其部分被隐藏时会自动回弹)
    https://github.com/rubensousa/RecyclerViewSnap
  • RecyclerView LayoutManager 实现风车效果
    https://github.com/Cleveroad/FanLayoutManager

RxJava

  • RxJava 绑定 Activity，Fragment 生命周期
    https://github.com/trello/RxLifecycle

ScoreView

  • SpiderWebScoreView是用于Android上的一个蛛网评分控件

SearchBar

  • https://github.com/mancj/MaterialSearchBar

SideBar

Splash

TextView
