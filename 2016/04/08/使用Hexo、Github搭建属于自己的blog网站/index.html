写作命令

  • 建立新文章：hexo n “新文章名”
  • 预览文章：hexo s
  • 生成网页：hexo g
  • 发布文章：hexo d
  • 生成网页并发布文章：hexo d -g

可能遇到的错误

  • 标签页点击后页面为空

    2. 检查对应tag文件夹中md文件是否添加type: “XXX”
